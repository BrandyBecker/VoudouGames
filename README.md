# VoudouGames
## RPG Dungeon Crawling Game made in React + Redux
--------------------------------------------------------

## Three Classes
| warrior |  ranger  | wizard   |
| :------ | :------- | :--------|
| melee | ranged | mage   |
| iron armor |   tunic |  robes |

--------------------------------------------------------

## Attributes:
- stamina ( health )
- strength ( warrior )
- agility ( ranger )
- intellect ( wizard )
